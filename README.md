# Basic Chatbot using Python

## Project Overview

This project is a simple rule-based chatbot developed using Python. The chatbot interacts with users through predefined responses and demonstrates fundamental programming concepts such as functions, conditional statements, loops, and user input/output.

## Objective

The objective of this project is to build a basic conversational chatbot that can respond to common user inputs such as greetings, questions about its status, expressions of gratitude, and conversation-ending commands.

## Features

* Responds to greetings such as "hello", "hi", and "hey"
* Answers basic questions like "how are you"
* Provides information about itself
* Responds to "thank you" messages
* Ends the conversation when the user enters "bye" or "goodbye"
* Handles unknown inputs with a default response

## Technologies Used

* Python 3
* Functions
* If-Elif Statements
* While Loops
* Input and Output Operations
* String Manipulation

## How to Run

1. Install Python 3 on your system.
2. Open the project folder in VS Code.
3. Open the terminal.
4. Run the following command:

python chatbot.py

If Python is installed using the Windows launcher:

py chatbot.py

## Sample Interaction

Welcome to Basic Python Chatbot

You: hello

Bot: Hello! How can I help you today?

You: how are you

Bot: I'm doing well, thank you for asking. How about you?

You: thanks

Bot: You're welcome! Happy to help.

You: bye

Bot: Goodbye! Have a great day.

## Learning Outcomes

Through this project, the following concepts were practiced:

* Creating and using functions
* Implementing decision-making using if-elif statements
* Using loops for continuous interaction
* Taking user input and displaying output
* Building a simple rule-based chatbot

## Conclusion

This project demonstrates the basic working principle of a chatbot using predefined rules. It serves as an introduction to conversational applications and provides a foundation for developing more advanced AI-powered chatbots in the future.
